Game Overview

Ludoteca is a strategic puzzle game that challenges you to place blocks while following unique and evolving rules. Some levels require connecting same-colored blocks, while others forbid them from touching. With each puzzle, you must think ahead and adapt to new constraints, putting your logic and creativity to the test in a dynamic and engaging experience.

How to Play

Controls:

  • Drag and drop blocks on the game board
  • Use right mouse button to rotate blocks
  • Pro tip: You can rotate while dragging!

Each level presents new challenges and rules. Pay attention to the color connections and block placement restrictions. Strategic thinking and careful planning are key to success!

Game Features

  • Progressive Difficulty
  • Intuitive Controls
  • Dynamic Puzzles
  • Creative Problem Solving
